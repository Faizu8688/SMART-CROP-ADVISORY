# 🌱 Smart Crop Advisory System with Chatbot

A data-driven Smart Crop Advisory System that provides intelligent crop recommendations based on soil and environmental conditions. The system also includes an interactive chatbot to assist farmers with real-time guidance.

---

## 🚀 Features

- 🌾 Crop recommendation based on soil nutrients
- 🤖 Integrated chatbot for farmer assistance
- 🌍 Multi-language support
- 📊 Dashboard for user interaction
- 🌱 Fertilizer recommendation system
- 📍 District-based agricultural data
- 🔐 User authentication system

---

## 🛠️ Tech Stack

- Python
- Flask
- HTML, CSS
- Jinja2 Templates
- SQLite Database
- JSON Data Handling

## ⚙️ Installation & Setup

1. Clone the repository:


git clone https://github.com/Faizu8688/SMART-CROP-ADVISORY.git


2. Navigate into the project folder:


cd SMART-CROP-ADVISORY


3. Install dependencies:


pip install -r requirements.txt


4. Run the application:


python app.py


5. Open in browser:


http://127.0.0.1:5000
